# Multi-Page Web Project

This project is a simple multi-page website created for practice using HTML and CSS.

## Pages Included
- Home Page
- Task 1 Page
- Task 2 Page

## Features
- Clean and responsive layout
- Structured page navigation
- Modern UI design
- Practice of HTML elements and CSS styling

## Technologies Used
- HTML5
- CSS3

## Purpose
The purpose of this project is to improve frontend development skills by building multiple connected web pages and practicing layout design, styling, colors, spacing, and navigation.

## Learning Outcomes
Through this project, I practiced:
- Page structure using HTML
- Styling with CSS
- Layout alignment and spacing
- Creating reusable design patterns

## Future Improvements
- Add JavaScript interactivity
- Improve responsiveness for mobile devices
- Enhance animations and transitions
